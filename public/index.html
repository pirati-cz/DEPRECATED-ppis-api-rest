API

To list users use https://example.com/users

To list groups use https://example.com/groups

To list permissions use https://example.com/permissions

You can search for users, groups and/or permissions by using these urls:

  • https://example.com/user/USERNAME
  • https://example.com/group/NAME
  • https://example.com/permission/NAME

Relations can be gathered like this:

  • https://example.com/user/USERNAME/groups
  • https://example.com/user/USERNAME/permissions
  • https://example.com/group/NAME/members
  • https://example.com/group/NAME/permissions
  • https://example.com/permission/NAME/groups
  • https://example.com/permission/NAME/users
