API

To list users use https://example.com/users

To list groups use https://example.com/groups

You can search for users and/or groups by using these urls:

  • https://example.com/user/USERNAME
  • https://example.com/group/NAME

User-group relations can be gathered like this:

  • https://example.com/user/USERNAME/groups
  • https://example.com/group/NAME/members
